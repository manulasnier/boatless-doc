# Mixins

boatless mixins are split into four categories. Import `~boatless` once and they're all available.

| Category | File | Description |
|----------|------|-------------|
| [Effects](./effects) | `_mixins.less` | Shadows, transitions, transforms, gradients, border-radius |
| [Layout](./layout) | `_layout.less` | Flex, grid, column layout, centered container |
| [Typography](./typography) | `_typography.less` | Font stack shorthand |
| [Forms](./forms) | `_forms.less` | Placeholder color |
